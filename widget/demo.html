Widget de Chat — Área de Teste

Esta página simula o site de um cliente final com o widget TUUVO embutido. Configure os parâmetros abaixo e clique no balão roxo no canto inferior direito.

📋 Como testar

  • Crie um Tenant via painel do Superadmin
  • Crie uma Conexão de Canal do tipo webchat
  • Crie um Widget no Widget Builder e publique
  • Cole os IDs nos campos abaixo e clique em Aplicar
  • O balão de chat aparecerá no canto inferior direito

⚙️ Configuração do Widget

Preencha os IDs do tenant e do widget criados no painel.

🔗 Snippet de Integração

Cole este código no HTML do site do cliente para embutir o widget:

<script src="https://api.tuuvo.app.br/tuuvo-widget.js" data-tenant="SEU_TENANT_ID" data-widget="SEU_WIDGET_ID" data-backend="https://api.tuuvo.app.br" data-socket="https://api.tuuvo.app.br"></script>
